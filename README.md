## 🚀 What princeps does

princeps is a desktop app for running project work in a simple day-by-day flow. It helps you turn a project into clear tasks, track them on a kanban board, and launch work with a manual /execute trigger.

It is built for Claude Cowork use, but it works well as a general project planner for people who want a clear view of what to do next.

## ✨ Main features

- Day-by-day plans that break work into small steps
- Kanban dashboard with pixel-style characters
- Auto-scheduled tasks based on your plan
- Manual /execute trigger for direct task runs
- Simple task view for checking what is ready, active, or done
- Clean layout that keeps your work in one place

## 📋 How to use princeps

### 1. Set up a project

Start by creating a project name. This gives your work a place to live inside the app.

### 2. Build a day-by-day plan

Add tasks in the order you want them to run. Keep each task small so it is easy to track.

### 3. Watch the kanban board

Use the board to see what is waiting, what is active, and what is done. The pixel characters make the board easier to scan.

### 4. Run a task

Use the manual `/execute` trigger when you want to start work right away. This is useful when you do not want to wait for the next scheduled step.

### 5. Let auto-scheduling work

Princeps can place tasks on a schedule so your plan moves forward without constant manual setup.

## 🧠 Best way to organize your work

For the clearest results:

- Keep task names short
- Use one goal per task
- Put the most important work at the top
- Review the plan before you start the day
- Mark finished tasks as done right away

A simple plan works better than a long one. Small steps are easier to follow and easier to fix.

## 🔒 Local use and privacy

princeps is meant for local project execution and task tracking. Your plans and board state stay tied to your own setup unless you choose a shared workflow elsewhere.

This makes it a good fit for people who want a focused tool without a lot of extra setup.

## 🛠️ Troubleshooting

### The app will not open

- Check that the file finished downloading
- Try opening it again from the extracted folder
- Right-click the app and choose Run as administrator
- Make sure your Windows version is up to date

### Windows says the app is blocked

- Open the file’s properties
- Look for an Unblock option if Windows shows one
- Try the newest release again from the download page

### The app opens but looks empty

- Create a project first
- Add at least one task
- Refresh the board view if the app has that option

### Tasks do not run

- Check that the task is ready to execute
- Try the manual `/execute` trigger again
- Make sure the task is not already marked done
